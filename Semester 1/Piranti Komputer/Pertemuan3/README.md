# Pengenalan Processor Komputer

## Pengertian Processor

- Processor merupakan salah satu komponen paling utama dari rangakaian komputer, karneanprocessor merupakan otak komputer.

## Serajarh Processor

- **1971**, munculah microprocessor pertama inter yaitu microprocessor 4004 digunakan pada mesin penghitung buscom.
- **1972**, intel merilis microprocessor 8008 dengan kekuatan 2 kali lipat dari pendahulunya.
- **1974**, microprocessor 8008 menjadi otak pertama dari sebuah komputer bernama Altair.
- **1978**, microprocessor 8008 merupakan processor yang digunakan pada komputer pribadi.
- **1982**, intel 286 adalah sebuah processor yang pertama kadi dapat menggunakan softwate yang digunkan unutk processor sebelumnya.
- **1985**, intel 386 memiliki 275000 transistor yang tertanam di processor.
- **1989**, Processor yang pertama kali memudahkan berbagai aplikasi yang tadinya harus mengetikan command-command menjadi hanya sebuah klik saja.
- **1993**, processor generasi baru yang mampu menganagi berbagai jedis data seperti suara, bunyi, tulisan tangan, dan foto.
- **1995**, Processor yang dirancang untuk digunakan pada aplikasi server mempunyai 5.5jt transistor.
- **1997**, Processor Pentium II dirancang secara khusus unutk mengolah data video, audio, dan grafik sehingga memiliki 7-5 juta transistor.
- **1998**, Intel Pentium II Xeon merupakan processor yang dibuat untuk kebutuhan pada aplikasi server.
- **1999**, Processor Intel Celeron merupakan processor yang ditunjukan untuk pengguna yang tidak terlalu membutuhkan kinerja processor.
- **1999**, Processor Pentium III merupakan processor yang diberi tambahan 70 intruksi baru sehingga mampu menampilkan tiga dimensi, audio streaming, dan aplikasi-aplikasi lainnya.
- **1999**, Intel mengeluarkan seri Xeon dengan jenis Pentium III yang mempunyai 70 perintah SIMD yang dapat mempercepat pengolahan informasi.
- **2000**, Processor Pentium IV merupakan produk intel yang memiliki kecepatan hingga *3.06GHz* dengan formafactor pin 478.
- **2001**, Processor Intel Pentium 4 Xeon merupakan processor khusus sebagai computer server dengan pin lebih banyak dari processor intel pentium 4.
- **2001**, Itanium adalah processor pertama basis 64 bit, Processor ini sudah dibuat dengan struktur yang didasarkan pada design dan teknologi Intel's Expicitly Parallel Instruction Computing.
- **2002**, Intel merilis Itanium 2 merupakan generasi kedua dari keluarga itanium.
- **2003**, Intel Pentium M Processor menggunakan cipset 855, processor ini sudah digunakan pada komputer yang mudah dibawa kemana-mana.
- **2004**, Intel E7520/E720 dilengkapi dengan cipset 855 dengan fitur baru 2MB L2 Cache 400MHz system bus dan soket processor dengan seri-seri Pentium M sebelumnya.
- **2005**, Intel Pentium D820/830/840 adalah processor berbasis 64 bit, disebut dual core karena menggunakan 2 buah inti dengan konfigurasi 1MB L2 cache pada setiap core.
- **2006**, Intel Core 2 Quad Q6600 merupakan Processor unutk tipe desktop yang memiliki 2 buah core dengan configurasi 2.4GHz dengan 8MB L2 Cache.
- **2006**, Intel Quad-core Xeon X3210/3220 merupakan processor yang digunakan untuk tipe server memiliki 2 buah core dengan masing-masing memiliki konfigurasi 2.13 dan 2.4GHz dengan 8MB L2 cache.

## Fungsi Processor

- Dapat memproses data yang diterima dari input, kemudian akan menghasilkan pengeluaran beruapa output.
